File: 6_Image_Adaptation.html

<picture>
  <!-- Large screens get a wide image -->
  <source
    media="(min-width: 800px)"
    srcset="https://picsum.photos/1000/300?random=1"
  >

  <!-- Medium screens get a medium image -->
  <source
    media="(min-width: 500px)"
    srcset="https://picsum.photos/700/300?random=2"
  >

  <!-- Small screens get a smaller image -->
  <img
    src="https://picsum.photos/400/300?random=3"
    alt="Landscape"
    style="width: 100%; border-radius: 8px;"
  >
</picture>

<pre>
    <h3>Explanation</h3>
&lt;picture&gt; allows you to provide different images based on viewport width.
&lt;source&gt; with media acts like a conditional rule.
Viewports 800px or wider get the 1000px wide image.
Viewports 500–799px get the 700px wide image.

The &lt;img&gt; element at the end is the fallback for viewports smaller than 500px.
Small displays download a smaller image, which improves performance on mobile.
</pre>

<style>
.container {
  display: flex;
  flex-wrap: wrap;      /* allows items to wrap onto new lines */
  gap: 10px;
  background: #e0e0e0;
  padding: 10px;
}

.item {
  background: #2196F3;
  color: white;
  padding: 20px;
  flex: 1 1 150px;      /* grow, shrink, base width 150px */
  text-align: center;
}

/* Change order on small screens */
@media (max-width: 600px) {
  .item:nth-child(3) { order: -1; }  /* moves item 3 to the front */
}
</style>

<div class="container">
  <div class="item">One</div>
  <div class="item">Two</div>
  <div class="item">Three</div>
  <div class="item">Four</div>
</div>
<h3>Explanation</h3>
<pre>
<b>flex-wrap: wrap</b> → items wrap to the next row if they don’t fit

<b>flex: 1 1 150px</b> → items can grow, shrink, and start with 150px width

<b>order</b> → lets you change the visual order without changing HTML

<b>Media query - .item:nth-child(3) { order: -1; }</b> shows responsive behavior on small screens moves item 3 to the front

</pre>
